# 同城速送微信小程序原型

这是一个参考“同城即时配送/跑腿下单”业务的小程序 MVP 原型，品牌与界面已改成自有名称“同城速送”，避免直接复制第三方品牌。

## 已完成页面

- 首页：城市、地图氛围卡、服务类型、发货/收货地址、快捷服务、下单入口
- 地址选择：搜索、历史地址、发货/收货地址带回首页
- 确认订单：物品类型、重量、备注、费用预估、模拟支付下单
- 订单详情：状态流转、配送地图氛围卡、订单信息、模拟推进
- 订单列表：全部/进行中/已完成筛选
- 消息中心：订单提醒、优惠活动、保障通知
- 我的：会员信息、优惠券、常用服务、骑手/后台入口占位

## 如何运行

1. 打开微信开发者工具。
2. 选择“导入项目”。
3. 项目目录选择当前文件夹：`/Users/Admin1/Documents/Codex/2026-07-09/xian`
4. AppID 可选择测试号，或继续使用 `project.config.json` 中的 `touristappid`。
5. 编译后在首页点击“送到哪里？”选择地址，再点击“去下单”。
